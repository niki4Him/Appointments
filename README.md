# Laravel Appointment Application

A Laravel-based application for managing appointments with features like CRUD operations, API endpoints, and user authentication using Sanctum.

## Requirements

- PHP >= 8.1
- Composer
- Node.js & npm
- MySQL Database
- Laravel 11
